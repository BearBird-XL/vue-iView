# Vue学习笔记
## Vue基础概念
- 特点
1. 采用**组件化**模式, 代码复用率高
2. **声明式**编程, 简化代码
3. 使用**虚拟DOM + 优秀的Diff**算法, 尽量复用DOM节点


## 使用Vue

- CDN引入
```javascript
<script src="https://unpkg.com/vue@next"></script>
```
